# 📘 GitLearn — אתר לימוד GitHub בעברית

אתר אינטראקטיבי ללימוד Git ו-GitHub מאפס ועד רמה מקצועית, בעברית מלאה.

---

## 🚀 תכונות

- **טרמינל מונפש** — הדגמה חיה של פקודות Git
- **15 פקודות מוסברות** — עם דוגמאות קוד לכל אחת
- **פילטר לפי קטגוריה** — בסיסי / ענפים / Remote / מתקדם
- **GitHub Flow** — תהליך עבודה מקצועי ב-5 שלבים
- **חידון אינטראקטיבי** — 5 שאלות עם ניקוד ומשוב
- **Dark / Light Mode** — מתג ערכת נושא

---

## 📁 מבנה הפרויקט
```
gitlearn/
├── index.html   # מבנה הדף — תוכן וסמנטיקה
├── style.css    # עיצוב — צבעים, פריסה, אנימציות
└── script.js    # לוגיקה — טרמינל, חידון, פילטרים
```

---

## ▶️ הרצה מקומית
```bash
git clone https://github.com/your-username/gitlearn.git
cd gitlearn
# פתח את index.html בדפדפן — אין צורך בשרת
open index.html
```

## 📚 תוכן הלימוד

1. **מבוא** — מה זה Git ו-GitHub והקשר ביניהם
2. **פקודות** — `init`, `clone`, `add`, `commit`, `push`, `pull`, `branch`, `merge`, `stash`, `rebase` ועוד
3. **תהליך עבודה** — GitHub Flow מקצועי
4. **חידון** — בדיקה עצמית עם משוב מיידי

---

## 🤝 תרומה לפרויקט

Pull Requests יתקבלו בברכה!
```bash
git switch -c feature/your-idea
# בצע שינויים
git commit -m "feat: תיאור השינוי"
git push origin feature/your-idea
# פתח Pull Request ב-GitHub
```
